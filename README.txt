Publish blog posts to multiple twitter accounts.

== Description ==

* Publish blog posts to **one or more** twitter accounts.
* **Shorten links** to your blog post with bit.ly (optional).
* Users can specify their twitter details when editing their profile.
* Easily **customize** the twitter message.
* Decide which **user levels** can add their twitter details or publish a blog post to twitter.

Feature requests and bug reports: jeroen.bolle@gmail.com

== Installation ==

1. Download the plugin's archive and extract it with WinRAR.
2. Place the 'twimp-wp.php' file in your wordpress plugin directory (wp-content/plugins).
3. Go to the plugins management page in the wordpress adminstrator area and activate the twimp plugin.
4. Browse to the twimp settings page (Settings -> Twimp) and configure the plugin (optional).

== Frequently Asked Questions ==

= Is a bit.ly account required for this plugin? =

No, you can disable the bit.ly service on the twimp settings page. It is enabled by default with a general account, maintained by twimp. To monitor traffic on your bit.ly links, you should create your own account on bit.ly and receive an API key.
